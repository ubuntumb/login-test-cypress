# login-test-cypress
## Proyecto de Pruebas E2E con Cypress v14 + TypeScript + Mochawesome

#### Este proyecto implementa pruebas end-to-end (E2E) automatizadas con 

**[Cypress 14](https://docs.cypress.io/)** y **[TypeScript](https://www.typescriptlang.org/)**. 

- El patrón utilizado es **Page Object Model (POM)** para mantener el código modular, reutilizable.

- Las pruebas se enfocan en el **formulario de login** de la web [Practice Test Automation](https://practicetestautomation.com).

#### Script disponibles

- npm run test             # Ejecutar todas las pruebas
- npm run cy:open          # Abrir interfaz de Cypress
- npm run test:chrome      # Ejecutar pruebas en navegador Chrome
- npm run test:firefox     # Ejecutar pruebas en navegador Firefox
